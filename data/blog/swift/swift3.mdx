### 📌 String (Matn)

**String** – oddiy matn. Uni qo‘shtirnoq ichida yozamiz:

```swift
let myFirstItem: String = "Hello world"
```

_Turi aniq yozish (`: String`) ixtiyoriy, lekin yangi o‘rganayotganda foydali._

📝 **E’tibor bering:**  
✔️ String har doim qo‘shtirnoq ichida bo‘lishi kerak.  
✔️ Agar qo‘shtirnoq ishlatilmasa, kompilyator xato beradi.  
✔️ Kod yozayotganda, matn to‘q sariq rangda ko‘rinadi – bu uning String ekanligini bildiradi.  

### ✅ Boolean (rost/yolg‘on)

**Boolean** – faqat ikkita qiymatni qabul qiladi:

```swift
let isSwiftAwesome: Bool = true
let isBugFree: Bool = false
```

📌 **Noto‘g‘ri foydalanish:**

```swift
let myItem: String = "true"  // ❌ Xato!
```

✔️ Bool faqat `true` yoki `false` bo‘lishi kerak.  
✔️ Boolean o‘zgaruvchilari odatda shartli operatorlar (`if`, `while`) bilan ishlatiladi.  

### ⚠️ Swift va tur xavfsizligi

Swiftda tur xavfsizligi juda kuchli! Agar noto‘g‘ri turdagi qiymat berilsa, xatolik yuzaga keladi:

```swift
let myItem: Bool = "Hello world"  // ❌ Xato!
let anotherItem: String = true    // ❌ Xato!
```

✅ Agar tur aniq ko‘rsatilmasa, Swift o‘zi aniqlaydi:

```swift
let myItem = true  // Bool
tlet myText = "Hello world"  // String
```

### 📆 Date (Sanalar)

Swift-da sanalar bilan ishlash uchun `Date` turi mavjud:

```swift
let myDate: Date = Date()
```

📌 `Date()` – joriy sanani qaytaradi.  
✔️ Uni vaqtni saqlash, tahlil qilish va sanalarni solishtirish uchun ishlatamiz.  

### 🔢 Numbers (Raqamlar)

Swift-da sonlar quyidagi turlarga bo‘linadi:

```swift
let myFirstNumber = 1        // Int (butun son)
let mySecondNumber = 1.0     // Double (o‘nli kasr)
let myThirdNumber: CGFloat = 1.0  // CGFloat (UI uchun)
```

✔️ `Int` – faqat butun sonlar (1, 2, 10).  
✔️ `Double` – o‘nlik kasrli sonlar (1.5, 3.14).  
✔️ `CGFloat` – UI elementlar o‘lchami bilan ishlash uchun.  

📌 **Qoida:**  
- `Double` – matematik hisob-kitoblar uchun ishlatiladi.  
- `CGFloat` – UI dizayni uchun (masalan, rasm o‘lchami, shrift hajmi).
