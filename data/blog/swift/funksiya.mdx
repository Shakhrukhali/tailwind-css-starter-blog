## 🔥 Funksiyalar  

Hozirgacha shunday kod yozdik:  

```swift
var myItem: Bool = false
myItem = true
print(myItem)
```

Bu Playground uchun yaxshi, lekin haqiqiy ilovada kod funksiyalarga bo‘linadi.  

### ✅ Funksiya yozish:  

```swift
func myFirstFunction() {
    print("My first function called")
}
```

#### 📌 Tushuncha:  
- `func` – funksiya kalit so‘zi.  
- `myFirstFunction` – funksiya nomi.  
- `()` – parameterlar joyi (hozir bo‘sh).  
- `{}` – kod bloki (if shartlaridagi kabi).  

### ✅ Bu kod faqat chaqirilganda bajariladi:  

```swift
myFirstFunction()
```

#### 📢 Konsolda natija:  
🖥 *My first function called*  

## 🔄 Funksiyalar bir-birini chaqirishi mumkin  

```swift
func mySecondFunction() {
    print("My second function called")
}

func myFirstFunction() {
    print("My first function called")
    mySecondFunction()
}
```

#### 📌 Tushuncha:  
- `myFirstFunction()` chaqirilganda, avval o‘z kodi bajariladi, keyin `mySecondFunction()` chaqiriladi.  

### 🔗 Funksiyalar zanjiri:  

```swift
func myThirdFunction() {
    print("My third function called")
}

func mySecondFunction() {
    print("My second function called")
    myThirdFunction()
}

func myFirstFunction() {
    print("My first function called")
    mySecondFunction()
    myThirdFunction()
}
```

### ✅ Ilova ishlash jarayoni:  
1️⃣ `myFirstFunction()` chaqiriladi  
2️⃣ `mySecondFunction()` ishga tushadi  
3️⃣ `myThirdFunction()` ishlaydi  

## 🔐 Funksiyalar kirishni boshqaradi  

```swift
func getUsername() {
    let username = "Ali"
    print(username)
}

getUsername()
```

### 🔹 E’tibor bering!  
- `username` faqat funksiya ichida mavjud.  
- Agar tashqarida `print(username)` desak, xato beradi.  

### ✅ Agar tashqarida bo‘lsa:  

```swift
let username = "Ali"
func getUsername() {
    print(username)
}
```

## 🎯 Funksiyadan ma’lumot qaytarish  

```swift
func getUsername() -> String {
    let username = "Ali"
    return username
}

let name = getUsername()
```

#### 📌 Tushuncha:  
- `-> String` – funksiya `String` qaytarishini bildiradi.  
- `return` – qiymatni qaytaradi.  
- `name` endi `"Ali"` ga teng.
